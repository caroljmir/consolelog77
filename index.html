Projeto Console

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS.

O console serve para inspecionar, interagir com elementos de uma página, ou até mesmo para conferir se há mensagens de erros. Nesse projeto é necessário usar o console para dar respostas de códigos produzidos na parte da programção. Para abrir o console basta clicar com a parte direita do mouse, achar e clicar no botão inspecionar e depois clicar na área do console que mostrará o que foi cobrado no código.
